1. GET /news - получение списка всех новостей. Особенность - пагинация + для каждой новости должно быть указано количество комментариев к ней
2. GET /users - получение списка всех пользователей. Особенность - пагинация
3. GET /news/filter/...&...&... - получение списка новостей с фильтрацией по дате и категориям и автору + пагинация + указано кол-во комментариев
4. GET /users/{id} - получение юзера по id
5. GET /news/{id} - получение новости по id - комментарии к новости возвращаются в виде массива
6. POST /news - добавление новости
7. POST /users - добавление юзера
8. GET /news/categories - получение всех категорий
9. POST news/categories - создание категории
10. POST /news/{id}/comments - создание комментария
11. GET /news/{id}/comments - получение всех комментариев по новости
12. PUT /news/{id} - редактирование новости
13. PUT /news/{id}/comments/{id} - редактирование комментария
14. PUT /users/{id} - редактирование юзера
15. DELETE /users/{id} - удаление юзера
16. DELETE /news/{id} - удаление новости
17. DELETE /news/{id}/comments/{id} - удаление комментария
18. POST /news/categories/{id} - получение новостей по категории id
